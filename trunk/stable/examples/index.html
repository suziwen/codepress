CodePress

Online Real Time Syntax Highlighting Editor

CodePress is a code editor with syntax highlighting written in JavaScript that colors text in real time while it's being typed in the browser.

Visit codepress.fermads.net for updates.

Example shows syntax highlighting for Java, PHP, JavaScript and HTML. Other languages like PERL, Python, Ruby, C++ and C are on the way.
