CodePress

Editeur en ligne avec coloration syntaxique instantanée

CodePress est un éditeur de code source en ligne colorant syntaxiquement les données tappées par l'utilisateur.

C'est un système légé, extensible et simple (comparé aux projets similaire en ligne). Les performances ont été testées avec de gros fichiers (plus de 3000 lignes).

Visitez http://codepress.fermads.net/ pour les mises à jours.

Demo

Fonctionalités

Voici une liste exhaustive des fonctionalités de CodePress version Devel

  • Coloration syntaxique des languages : PHP, JavaScript, Perl, CSS, HTML, Java et SQL
  • Coloration syntaxique générique
  • Menu contextuel permettant de choisir le type de coloration
  • Auto-fermeture des caractères suivants : " , ' , ( , { et [
  • Insertion de codes prédéfinis par pression de la touche tabulation. Au niveau d'indentation courant sous Firefox. Exemples↓
    html + [tab] va insérer dans un code html
    <html></html>
    head + [tab] va insérer dans un code html
    <head>
        <meta http-equiv="content-type" content="text/html; charset=utf-8" />
        <title></title>
    </head>
    if + [tab] va insérer dans un code php
    if() { }
    func + [tab] va insérer dans un code php
    function () { }

    Pleins d'autres "raccourcis" sont à découvrir.
  • Replacement du curseur d'insertion après insertion d'un extrait d'un code prédéfini
  • Maintient du niveau d'indentation du code à chaque retour à la ligne. (sous Firefox uniquement pour l'instant)
  • Mode plein écran
  • Prise en charge d'un historique de modifications (utilisation des raccourcis traditionels [ctrl] + z pour annuler et [ctrl] + y pour rétablir)
  • Menu contextuel permettant l'activation de certaines fonctionalités
  • Prise en charge des gros fichiers
  • Raccourcis clavier personalisable. Exemples↓
    [ctrl]+[shift]+[espace] insert &nbsp;
    [ctrl]+[shift]+[enter] insert <br/>
